## ¿Qué es Typescript?

Es un lenguaje de programación de **[código abierto](https://github.com/microsoft/TypeScript) desarrollado por Microsoft** que se basa en JavaScript. 

**TypeScript** añade funcionalidades y herramientas adicionales a JavaScript, como la **verificación de tipos estáticos**, lo que significa que es posible revisar los errores en el código antes de ejecutarlo. 

Este lenguaje busca **facilitar el desarrollo de aplicaciones** grandes y complejas proporcionando una estructura de código más ordenada y predecible. 

Aprender TypeScript puede ayudar a **aumentar la calidad y la eficiencia del código**.
